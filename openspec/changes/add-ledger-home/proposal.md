# Change: 新增记账首页和统计功能

## Why

本项目要实现一个纯本地记账 App，用于记录日常支出和收入，并按月份、分类统计消费情况。

App 不需要联网，不需要登录，不需要云同步，也不需要请求服务端接口。所有账单数据都保存在用户本地设备中，使用 Room 数据库存储。

## What Changes

- 新增启动页 Activity，使用启动页设计图作为全屏背景，并在展示后进入主页面。
- 新增明细模块，展示账单列表、月份筛选、类型筛选、总支出和总入账。
- 新增“记一笔”功能，支持支出、入账、不计入收支三种记录类型。
- 新增本地 Room 数据库存储账单记录。
- 新增统计模块，按年月展示支出或入账汇总、分类占比、分类排行、每日对比和月度对比。
- 新增统计分类详情页，支持按金额和按时间查看分类下的账单明细。
- App 整体不请求网络，不依赖服务端接口。
- 代码里涉及分类图标、占位图片或临时图片时，统一先使用 Android 默认启动图标 `R.mipmap.ic_launcher`。

## Impact

- Affected specs: `ledger-home`
- Affected app area: 启动页、明细页、记账页面、日期选择、类型筛选、统计页、统计详情页、本地数据层
- Data storage: 仅使用 Room 本地数据库
- Network: 不需要网络访问
- Icon implementation: 默认使用 `R.mipmap.ic_launcher`
